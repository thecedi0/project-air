Apart from the above illustrations, there are other classes that works with the ad-btn to further design the button to your taste. The by Default was designed with shadows in it, to get rid of this, you need to add the class 'ad-flat' to the button. This ensures that the design of the button is completely comprehensive with flat design templates

Class Name Description Rendering
ad-btn Default style to call airDesign's button style Button
btn-[color] Default style to call airDesign's button style, with a pink background.
There are other colors to pick from like, .btn-prim, .btn-dark, .btn-yellow, .btn-tGreen, .btn-red, .btn-green, .btn-blue, .btn-white. 		Button
ad-flat By default, ad-btn has some shadows effect that make it like skeumophic.
If you prefer entirely flat design, just add the class ad-flat to your button and it will remove those effect 		Button
ad-block Add this class to remove the rounded edge of the buttons into a more blocky style Button
ad-round Add this class to have the rounded edge Button
outline Add this to ouline the button with the color from the btn-[color] and the text color will be the same
This makes the background of the button transparent and use the color of the button for both outline and font.
Remember that this effect works for all shapes and sizes of the button 		Button
clear Add this class to remove the background, and outline of the buttons. this is best used in a card.
When hovered, the text glows 		Button
