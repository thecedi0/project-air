By default, every paragraph element in the card has a padding of 16, and Header elements should have the class 'title'. Consider the example. Remember that ad-flat is used to remove all effects to a flat design. This card has a special class that helps it react to hover events. In case you want users to know that the card is clickable, you add the class 'hovable' to the card class.
The card comes with a transparent background, but we have predefined classes that changes the background-color with a matching text color for both divs and sections. This class can be used on almost all element. Examples of these classes are:

whiteBG --> gives a white background with a black text
darkBG --> gives a dark background with a white text
primBG --> gives the primary color background with a white text
blueBG --> gives a blue background with a white text
redBG --> gives a red background with a white text
yellowBG --> gives a yellow background with a white text
