Grid System

Like every css framework, there is the class responsible for arranging content by rows and columns. Twitter bootstrap always wraps all content in their class called "container" which is to center the content and "container-fluid" for a full width of the browser's width. airDesign also has its own container and its grid system.

We fir wrap every row and column in a container called "wrapper". Wrapper and wrapper-fluid does the same thing as the "container" and the "container-fluid" of the bootstrap. We personally advise that "wrapper-fluid" be not used for a website except you absolutely want to have the content at full with to the browser, like a slider and gallery. We find it fit to use wrapper instead for website so to center your contents irrespective of the resolution. 

                    
                     
                         <section>
                             <div class="wrapper">

                             </div>
                         </section>
                           
                           OR

                        <section>
                             <div class="wrapper-fluid">

                             </div>
                         </section>

In both instances, we have declared the wrapper and wrapper-fluid class to prepare the code for rows and colums. Right after the wrapper has been declered, the next class is ".ad-row" to create a row for the grid-system. Lets just assume the class wrapper as our container 

                    
                     
                         <section>
                             <div class="wrapper">

                                <div class="ad-row">
                                    
                                    Columns goes in here...

                                </div>
                             
                             </div>
                         </section>

This grid-system has a maximum of 12 columns just like bootstrap's. The full width columns is ad-cols-12 while for a 2 column grid will be half the size ad-cols-6. We illustrate example of a 3 Columns below. Rember that 3 divided by 12 will yeild 4. hence the class for a three column will be ad-cols-4 

                    
                     
                         <section>
                             <div class="wrapper">

                                <div class="ad-row">
                                    
                                    <div class="ad-cols-4">
                                    
                                    Column 1

                                    </div>
                                    <div class="ad-cols-4">
                                    
                                    Column 2

                                    </div>
                                    <div class="ad-cols-4">
                                    
                                    Column 3

                                    </div>

                                </div>
                             
                             </div>
                         </section>

There are countless possibilities and arrangements you can achieve with this. you could have nested columns but a column has to exits in a row. the naming conversion for our columns is not finalize. since ad-cols mean a small column, ad-coll means a large column etc. These are great the large screens, medium screens, small screens and xtra small screen devices. These column screen sizes could be mixed up for a perfect responsive design for each screen size. We will do our best to make it more robust and adoptive with other frameworks.
